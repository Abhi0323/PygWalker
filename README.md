```bash

```
## Using pygwalker

Import pygwalker and pandas to get started.

```python    
import pandas as pd
import pygwalker as pyg
```
## Import your data

```python
import seaborn as sns
flight = sns.load_dataset('flights')
x = pyg.walk(flight)
```
